## 🚀 如何使用

### 1. 启动后端API服务器
```bash
# 激活虚拟环境（如果有）
source .venv/bin/activate

# 启动API服务器
python main.py api --port 8000
```

### 2. 启动前端服务器
```bash
# 进入前端目录
cd frontend

# 启动HTTP服务器
python server.py --port 3000

# 或使用Python内置服务器
python -m http.server 3000
```

### 3. 访问应用
打开浏览器访问: **http://localhost:3000**
